Mac Studio — CrossOver & Parallels

A personal reference guide to running Windows applications on the Mac Studio M4 Max. Two solutions are in use: Parallels (full Windows virtual machine) and CrossOver (lightweight Windows compatibility layer). This document explains what each does, what is currently running on each, and the general rule for deciding which to use going forward.

What Each Solution Is

Parallels — Full Virtual Machine

Parallels runs a complete Windows 11 installation on your Mac. You get an entire Windows environment with its own desktop, file system, and settings. Windows apps run inside it exactly as they would on a real Windows PC. It is the safest compatibility option but carries more overhead since it boots and maintains a full OS.

CrossOver — Compatibility Layer

CrossOver (made by CodeWeavers, built on Wine) translates Windows application calls into Mac-native ones in real time, inside lightweight isolated containers called "bottles." No full Windows OS is required — it is faster to launch and uses fewer resources than a VM, but not every Windows app is compatible. Check the CrossOver compatibility database before committing a new app to it.

What Is Currently Running on Each

Vectric Aspire — via Parallels

Aspire is the 3D relief carving and CNC toolpath design tool, displayed full-screen on one monitor inside the Windows 11 VM. Parallels is the correct choice here: Aspire relies heavily on 3D viewport rendering and gadgets (such as the dog-bone fillet tool) that have known compatibility problems in CrossOver. Complex professional design software like this needs a fully faithful Windows environment.

Steam + Hogwarts Legacy — via CrossOver

Steam is installed inside a CrossOver bottle, with Hogwarts Legacy installed through it. CrossOver is the better choice for gaming on Apple Silicon because Parallels has known DirectX 12 issues that cause problems with this title. CrossOver handles it well — approximately 45 FPS at baseline, with frame generation nearly doubling that figure. Other Steam games can be added to the same CrossOver bottle.

QSPICE — via Parallels

QSPICE is a professional-grade SPICE circuit simulator made by Qorvo (created by Mike Engelhardt, the original author of LTspice). It is Windows-only and runs cleanly inside the Parallels Windows 11 VM. QSPICE is used for serious electronics simulation work including RC/RL transient analysis, frequency sweep and Bode plots, and AC/DC circuit behaviour. Download from qorvo.com and install it inside Windows exactly as you would on a real Windows PC. Save circuit files (.qsch) to your Mac Documents folder via the shared drive that Parallels makes visible inside Windows Explorer under "This PC."

General Rule — Which to Use Going Forward

  • Use Parallels for complex or professional applications (CAD, accounting, creative tools), anything that uses 3D viewports or plugins/gadgets, anything that needs full Windows OS access, and any app of unknown compatibility where you need a reliable safety net.
  • Use CrossOver for Steam games and PC gaming generally, self-contained apps that are confirmed on the compatibility list, and any situation where lower resource overhead matters.
  • Both can run simultaneously without conflict. The M4 Max with 36 GB RAM handles both running at the same time without performance issues — Aspire on one monitor in Parallels and a game in CrossOver on the other is a valid everyday setup.

Electronics Simulators

Three circuit simulators are in use, each suited to a different level of complexity and use case. The general progression is: start with Falstad for quick visual intuition, step up to EveryCircuit for interactive learning with proper Bode plots, and use QSPICE for serious SPICE-class analysis.

Falstad CircuitJS — Browser (No Install)

Falstad runs entirely in your Mac browser — no installation, no account, no cost. It is the best tool for quick visual sketching and building intuition about how a circuit behaves. Electricity is animated in real time: moving yellow dots show current flow, green indicates positive voltage, red is negative, and gray is ground. An oscilloscope probe can be attached to any wire with a right-click to show live waveforms.

Best for: quick sketches, visualising AC/DC coexistence, bias tee and RC/RL circuits where animated current flow helps understanding. Not suitable for Bode plots, precise frequency sweeps, or mixed-signal work.

  • How to run: open in any browser on your Mac — no setup required.
  • URL: falstad.com/circuit/circuitjs.html
  • Cost: free.

EveryCircuit — Browser or iOS App (No Install)

EveryCircuit is a step up from Falstad in analytical capability while remaining visually approachable. It runs natively on your Mac in two ways: as a web app in Chrome at everycircuit.com/app, or as the iOS app installed directly from the Mac App Store (your M4 Max supports iOS apps natively — no Parallels needed). It adds proper frequency sweep and Bode plot support, interactive component value sliders, and cloud save with sync across devices.

Best for: RC/RL filter characterisation, interactive exploration of component values, and work that benefits from animated visualisation combined with proper AC analysis. Ceiling is lower than QSPICE — complex mixed-signal or precision SPICE work will eventually need QSPICE instead.

  • How to run on Mac: browser at everycircuit.com/app (Chrome), or install iOS app from Mac App Store. No Parallels required.
  • Also available on: iPad and iPhone (synced via account).
  • URL: everycircuit.com
  • Cost: free with circuit size limits; full unlock ~$14.99 one-time (unlocks all platforms simultaneously).

QSPICE — Windows via Parallels

QSPICE is a full professional SPICE simulator — the same class of tool used by working electronics engineers. It was created by Mike Engelhardt (original author of LTspice) and is published free by Qorvo. It is Windows-only and runs inside the Parallels Windows 11 VM.

Best for: RC/RL time-constant transient analysis, frequency sweep and Bode plot generation, AC analysis of filter circuits, mixed-signal simulation, and any work that needs accurate SPICE-level mathematics. This is the right tool once Falstad or EveryCircuit feel limiting. QSPICE saves circuits as .qsch files and also accepts standard SPICE netlist .cir files.

  • How to run: start the Parallels Windows 11 VM, download the installer from the Qorvo website inside Windows, and install normally. No CrossOver needed — Parallels gives it a complete Windows environment.
  • Saving files: save .qsch project files to your Mac Documents folder via the shared drive visible in Windows Explorer under "This PC." This keeps circuit files on the Mac side, included in Time Machine backups.
  • SCPI integration: for automated test sequences, Python scripts on the Mac can drive the bench instruments (oscilloscope, signal generator, DMM, power supply) via SCPI/LXI while QSPICE runs the corresponding simulation — compare real bench results against the SPICE model.
  • URL: qorvo.com — QSPICE
  • Cost: free.
